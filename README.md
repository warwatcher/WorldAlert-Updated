# 🗺 WorldAlert

**WorldAlert** — Real-time global intelligence and OSINT dashboard.

Live situational awareness with an interactive world map, heatmaps, critical alerts, and geopolitical events.

## What It Is

A clean and powerful real-time dashboard that aggregates open-source intelligence, news feeds, and global events. Perfect for keeping **eyes on the world**.

**Key Features**:
- Interactive world map with live events and heatmaps
- Real-time alerts and critical notifications
- Multiple data layers (conflicts, migration, geopolitics, etc.)
- Dark, professional UI optimized for monitoring
- Fast updates and responsive design
